# Manga Explorer

Aplicación web para explorar y gestionar colección de mangas desarrollada con React y Zustand.

## Características

- Exploración de mangas con búsqueda y filtros
- Lista personal de favoritos
- Sistema de comentarios
- Autenticación de usuarios
- Interfaz responsive

## Tecnologías

- React 18
- Zustand (gestión de estado)
- Tailwind CSS
- React Router v6
- Axios

## Instalación

```bash
git clone [URL_DEL_REPOSITORIO]
cd VainillaGA_front_react_zustand
npm install
npm start
```

## Estructura

```
src/
├── components/  # Componentes React
├── store/      # Estado global
├── services/   # Llamadas API
├── utils/      # Utilidades
└── assets/     # Recursos
```

## Flujo de desarrollo

1. Añadir método en `apiService`
2. Crear/actualizar store con estado y acciones
3. Desarrollar componentes que consuman el estado
